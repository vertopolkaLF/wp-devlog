# Локализация WP DevLog

В этой директории хранятся файлы локализации для плагина WP DevLog.

## Файлы:
- `wp-devlog.pot` - шаблон для переводов
- `wp-devlog-ru_RU.po` - файл перевода на русский язык

## Компиляция MO файлов

Для компиляции PO файлов в MO, вы можете использовать:

1. Плагин Loco Translate в WordPress
2. Утилиту msgfmt из пакета GNU gettext
3. Или любую онлайн-утилиту для создания MO файлов

Команда для компиляции через msgfmt:
```
msgfmt -o wp-devlog-ru_RU.mo wp-devlog-ru_RU.po
```

Готовый MO файл должен иметь то же имя, что и PO файл, но с расширением .mo
